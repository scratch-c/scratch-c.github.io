1. 准备

  • github 账号一个

2. 创建仓库

创建一个 github 仓库，名称是你的用户名.github.io。
点最下面绿色的按钮。
点 add file,creat new file, 在 name your file 处写 index.html，点 commit changes，这就是你的主页。
参考：

<!DOCTYPE html>
<html>
<head>
  <title>scratch_szc's blog</title>
</head>
<body>
 <body>
  <h1>scratch_szc's blog</h1>
  <div id="bottem"></div>
</body>
</html>

如何写博客？ 还是点 add file,creat new file, 在 name your file 处写 blogs/test.md, 将 markdown 文件粘贴进去，点 commit changes，即可。等待一段时间后访问　你的 github 用户名.github.io/test 就能看到了。

3.美化

创建文件styles.css,写入如下代码：

/* ===== 浅色主题变量 ===== */

  text-decoration: none;

}

a:visited,
a:active,
a:focus {

}
